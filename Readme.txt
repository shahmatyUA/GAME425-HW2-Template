To use this code, use the following steps:

1) Create a blank 2D project in Unity Hub.
2) Allow Unity editor to initialize the project.
3) Save your blank 2D project and exit Unity.
4) Navigate to project directory in file system.
5) Drop the "Assets" and "Project Settings" folders into this directory.
6) Open and reload the Unity project you created.
7) In the project navigator, go to the assets directory
   and find the "Scenes" folder.
8) Click on the "BewareTheRavens" scene to load it.
7) Go to Unity->File->Build Settings and verify that
   the "BewareTheRavens" scene is "Scenes in Build" list.
8) If "BewareTheRavens" is not in the build list, click
   "Add Open Scenes". StationDefense should appear checked. 
9) The game should be ready for building and running.

Key Mappings: These can be changed by modifying the 
"Input Mapping Array" values in the Unity Inspector 
for the GameController Component of the similarly
named GC game object.

 Cast = Right 'CTRL'
 Hack = Return
 Jump = Spacebar
 Die  = Backspace (Hero Death Demo)
 AABB = 'S' (Box Indicator Demo)  
 Quit = 'ESC'
